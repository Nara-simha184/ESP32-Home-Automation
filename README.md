# ESP32 Home Automation System

## Overview
This project implements a Home Automation system using ESP32. 
It controls appliances based on sensor inputs and manual web interface commands.

## Components Used
- ESP32
- DHT11 Temperature Sensor
- LDR (Light Dependent Resistor)
- Ultrasonic Sensor (HC-SR04)
- 2-Channel Relay Module

## Features
- Automatic and Manual Mode
- Temperature-based fan control
- Motion-based light control
- Web interface control via WiFi

## Technologies Used
- Embedded C (Arduino IDE)
- ESP32 WiFi
- Sensor Interfacing
- Real-time control logic

## Project Outcome
Successfully implemented and tested a working prototype capable of controlling appliances based on environmental conditions and user inputs.
